Cube Masking Guide

A Mask uses sticker labels to modify the cube appearance.

How It Works

For each face of the cube, they follow the order URFDLB. Each of these faces are defined by the diagram below:

               +-------------+
               | UBL  UB  UBR|
               | UL   U   UR |
               | UFL  UF  UFR|
 +-------------+-------------+-------------+------------+
 | LUB  LU  LUF| FUL  FU  FUR| RUF  RU  RUB| BUR BU  BUL|
 | LB   L   LF | FL   F   FR | RF   R   RB | BL  B   BR |
 | LDB  LD  LDF| FDL  FD  FDR| RFD  RD  RBD| BDR BD  BDL|
 +-------------+-------------+-------------+------------+
               | DFL  DF  DFR|
               | DL   D   DR |
               | DBL  DB  DBR|
               +-------------+

Initial Mask

Applied Before Scramble

Functions like removing the stickers on a physical cube. Grey stickers can be moved. They can be used to train a substep. e.g. removing all of the side top layer stickers to train OLL, or removing all top layer stickers except the top cross to train ZBLS.

Applied After Scramble
